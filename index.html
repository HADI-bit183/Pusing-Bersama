Selamat Datang di Kalkulator SPL Eliminasi Gauss

Kalkulator ini membantu Anda menyelesaikan sistem persamaan linear (SPL) menggunakan metode Eliminasi Gauss secara langkah demi langkah. Anda dapat memasukkan persamaan linear dengan variabel bebas (misalnya x, y, z), dan kalkulator akan menampilkan proses eliminasi untuk mendapatkan solusi secara tepat.

Metode Eliminasi Gauss adalah cara sistematis untuk menyelesaikan sistem persamaan linear dengan mengubahnya menjadi bentuk matriks segitiga atas (upper triangular matrix). Metode ini sangat penting dalam aljabar linear dan berbagai aplikasi matematika terapan serta rekayasa.

Berikut adalah langkah-langkah utama dalam metode Eliminasi Gauss:

  1. Pilih elemen pivot di setiap kolom (biasanya diagonal utama) yang tidak nol.
  2. Jika elemen pivot bernilai nol, lakukan pertukaran baris untuk mendapatkan pivot bukan nol.
  3. Bagi baris pivot sehingga elemen pivot menjadi 1 (normalisasi pivot).
  4. Eliminasi elemen-elemen di bawah pivot dengan mengurangkan kelipatan baris pivot dari baris lain sehingga membentuk matriks segitiga atas.
  5. Setelah matriks berbentuk segitiga atas, lakukan substitusi balik untuk mencari nilai variabel secara bertahap.

Kalkulator ini menampilkan proses eliminasi secara lengkap dan langkah demi langkah, termasuk matriks awal, operasi baris yang dilakukan, serta hasil akhir solusi. Dengan demikian, pengguna dapat memahami bagaimana solusi sistem persamaan linear diperoleh secara transparan dan akurat.

Masukkan Sistem Persamaan Linear Anda

Gunakan variabel bebas dan tanda sama dengan (=), tiap persamaan dipisah baris baru.
